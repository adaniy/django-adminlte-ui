django-adminlte 是一个基于adminlte的django后台管理系统主题

adminlte 版本: 2.4.18

> 从 v2.0.0 版本开始, django-adminlte移除了所有除django以外的依赖包.

老版本(1.x)文档: [点击](https://django-adminlte-ui.readthedocs.io/)

## 安装

```
pip install django-adminlte-ui
```

## 设置

```
# settings.py

INSTALLED_APPS = [
    'adminlteui',
    'django.contrib.admin',
    ...
]
```
